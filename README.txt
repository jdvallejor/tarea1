Para ejecutar:
    - npm install
	- Listar cursos: ejecutar comando "node cursos.js"
	- Incribir cursos: ejecutar comando "node cursos.js inscribir --nombre=[nombreEstudiante] --id=[idCurso] --cedula=[cedulaEstudiante]"
		NOTA: reemplazar los corchetes por el valor correspondiente
